# vue-parcel

A vue-cli template for a VueJS project that's bundled using [Parcel][1].

## Usage

_Note: Make sure to install [vue-cli][2] first._

```bash
$ vue init arnellebalane/vue-parcel project-name
$ cd project-name
$ npm install
$ npm start
```
